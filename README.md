# [トレーニングの準備](prep/README.md)

準備は初日に1回だけ実施すればOKです。以降は、ご準備いただいたMicrosoftアカウントで、[Azure portal](https://portal.azure.com/)や、[演習環境 - Learn on Demand LMS (LODS)](https://tsfb.learnondemand.net/) にアクセスできます。
